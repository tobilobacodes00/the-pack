The Pack

How it all fits together

One command goes down; one event comes back up to the same screen. Qwen Cloud is the brain, our compute runs on Alibaba Cloud (ECS), and the data lives on managed Neon (Postgres) and Upstash (Redis). Follow the numbers 1→6 down the write path and back up the read path — it always returns to the frontend.

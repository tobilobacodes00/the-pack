The Pack  ·  System architecture

One command down, one event back up

A single round-trip: the browser sends a command, the backend writes exactly one event to the database, and that event streams back to the same browser. Qwen Cloud is the brain; our compute runs on Alibaba Cloud (ECS); the data lives on managed Neon (Postgres) and Upstash (Redis).
